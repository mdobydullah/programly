# Programly
Code Smarter, Solve Faster, Secure Stronger

## Contributing

1. Clone this repository

```bash
git clone https://github.com/mdobydullah/programly
```

2. Install dependencies

```bash
npm install
```

3. Start dev server

```bash
npm run dev
```
